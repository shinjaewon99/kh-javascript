이벤트 종류

load : 웹페이지의 정보(css, javascript , 이미지 등)가 로드 완료된 시점에발생 focus : 엘리먼트가 포커스를 받았을 때 발생 blur : 엘리먼트가 포커스를 잃었을 때 발생 reset : 리셋버튼을 눌렀을 때 발생 submit : 제출(submit)버튼을 눌렀을 때 발생
fullscreenchange : 전체화면 모드로 전환되거나 다시 일반모드로 전환 될때 발생 resize : 브라우저 창의 크기를 조절할 때 발생 scroll : 스크롤을 동작 했을 때 발생 cut : 잘라내기 작업을 했을 때 발생 copy : 복사 작업을 했을 때 발생
paste : 붙여넣기 작업을 했을 때 발생 keydown : 임이의 키보드가 눌렸을 때 발생 keypress : 쉬프트 ,Fn , CapsLock을 제외한 키가 눌린상태일 때 발생(연속적으로 발생) keyup : 키 누룸이 해제됐을 때 발생 mouseenter
mouseover mousemove mousedown mouseup click
dblclick contextmenu whell mouseleave mouseout
select dragstart drag dragend dragenter
dragover dragleave drop
